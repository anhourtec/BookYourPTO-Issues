# BookYourPTO Issues

Public issue tracker for [BookYourPTO](https://bookyourpto.com) — all-in-one leave, time tracking, expense, and document management for modern teams.

## Before You Open an Issue

- **Search first** — check [existing issues](https://github.com/anhourtec/BookYourPTO-Issues/issues) to avoid duplicates.
- If you find a matching issue, add a thumbs-up or additional context instead of opening a new one.

## Reporting a Bug

- Describe the problem with as much detail as possible.
- Include steps to reproduce, expected vs. actual behavior, and screenshots if applicable.
- **Do not post confidential information** (account credentials, personal data, etc.) — this is a public tracker.

## Requesting a Feature

- Describe the problem you're trying to solve and how you'd like it to work.
- Search existing issues first — if someone has already requested it, give it a thumbs-up to help us prioritize.
- We review feature requests regularly and prioritize based on community interest and product direction.

## Confidential or Urgent Issues

For anything sensitive or time-critical, email **support@anhourtec.com** directly. Reference the ticket number if a related public issue exists.
